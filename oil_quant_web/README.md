# 油价量化策略网页（V3）

这是一个把你提供的 `v3.py`（建模、出分层预警、回测、SHAP 影响因素）封装成 Web API 的前后端项目，并提供一个可视化网页。

## 运行方式（Windows / PowerShell）
1. 打开该目录的终端：`C:\Users\24714\oil_quant_web`
2. 执行：`.\run.ps1`
3. 浏览器打开：`http://localhost:8000`

## 前端使用
- 上传 CSV：要求至少包含 `Date`、`WTI_Crude`，以及 `v3.py` 用到的其它列（如 `Brent_Crude/SP500/USD_Index/VIX/OVX/Gold` 等）。
- 点击 `运行策略` 后会显示：
  - 当前信号（红/黄/绿）与对冲建议
  - 回测指标（累计收益、年化、最大回撤、夏普）
  - 资金曲线图与分层预警信号图
  - SHAP 影响因素 Top N
  - 买卖点执行时点（回测进入/退出）

## 说明
- 第一次运行可能较慢：包含 XGBoost 训练与 SHAP 解释。
- 若你不想每次都训练，可告诉我，我可以再加缓存/异步任务（提升体验）。
